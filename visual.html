Message Routing Decision Flow

This is the logic the product should run before every outbound message.

Cost guardrail
1. Need to send Order update, OTP, support reply, or marketing campaign.
2. User window open? If user messaged within 24 hours, answer as Service when possible.
3. Free entry point? If eligible ad/Page CTA started the chat, use the 72-hour free window.
4. Template needed? Outside windows, use approved templates only.
Service Support/sales reply inside customer service window. Lowest cost.
Utility Transactional, non-promotional, requested or expected update.
Authentication OTP, login, recovery, or transaction verification only.
Marketing Promotional or re-engagement message. Needs strict budget and opt-in.

Karzoun-Style AI Business Model

WhatsApp-first SaaS with AI text replies, voice-note replies, human handoff, and transparent usage billing.

New direction
1. WhatsApp message Text, template reply, media, or customer voice note arrives by webhook.
2. Voice to text If audio, download media, transcribe, and save the transcript.
3. AI agent Use client knowledge base, conversation history, CRM/order tools, and guardrails.
4. Approval check Auto-send simple answers. Ask a human for refunds, complaints, low confidence, or sensitive actions.
5. Reply format Send text for lower cost and speed, or generate a voice reply when enabled.
6. WhatsApp send Send through Cloud API and log service-window/category cost.
7. Dashboard Show AI ownership, transcript, source context, handoff state, Meta cost, and AI usage.
8. Billing Monthly plan + Meta pass-through + AI text/voice usage + setup services.
Platform subscription Seats, channels, inbox, automation, reports, API access, and support level.
Meta pass-through Official WhatsApp fees shown separately by country, category, and free/paid status.
AI usage add-on Charge for AI text replies, transcription minutes, generated voice minutes, and storage retention.
Services revenue WhatsApp onboarding, template setup, AI training, custom automation, and reseller support.

What Makes the Plan Correct

These checks keep the strategy aligned with pricing, policy, and market reality.

Validation
Transparent fees Meta rates are pass-through. Margin comes from fixed platform value.
